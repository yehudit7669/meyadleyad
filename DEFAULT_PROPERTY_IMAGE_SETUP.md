# הוראות להוספת תמונת ברירת מחדל לנכסים

## מה בוצע?
עדכנתי את הקוד כך שכל נכס שמוצג באתר **ללא תמונות** יציג תמונת ברירת מחדל של בית (התמונה שצורפה בשיחה).

## קבצים שעודכנו:
1. ✅ **AdDetails.tsx** - דף פרטי הנכס המלא
2. ✅ **AdCard.tsx** - כרטיס הנכס ברשימות (דף הבית, חיפוש וכו')
3. ✅ **SEO.tsx** - תגי מטא לשיתוף ברשתות חברתיות
4. ✅ **CityPage.tsx** - דף העיר עם רשימת נכסים
5. ✅ **BrokerProfile.tsx** - פרופיל המתווך עם הנכסים שלו

## איך זה עובד?
- **אם למשתמש יש תמונה** → תוצג התמונה שלו ✓
- **אם אין תמונה בכלל** → תוצג תמונת ברירת המחדל (התמונה המצורפת) ✓

## בדיקה:
לאחר העתקת התמונה:
1. הרץ את האפליקציה (`npm run dev`)
2. צור נכס חדש **ללא העלאת תמונות**
3. בדוק שתמונת ברירת המחדל מוצגת בכרטיס הנכס ובדף הנכס המלא
